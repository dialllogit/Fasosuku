## Guide de Conversion et Préparation

### 🎯 Résumé Exécutif

**FasoSuku** est une Progressive Web Application (PWA) innovante conçue comme une marketplace et une liseuse numérique sécurisée pour les supports académiques maliens.

**Problème :** Les ressources pédagogiques au Mali ne sont pas monétisées, massively piratées, et inaccessibles financièrement.

**Solution :** Micro-paiement mobile (500 FCFA/chapitre), protection anti-piratage (watermark + Canvas), et revenus directs pour les enseignants (80%).

### Méthode 2 : Pandoc
```bash
pandoc docs/PROJECT_REPORT_FR.md -o Rapport_FasoSuku.docx \

  --number-sections
```

### Méthode 3 : Copier-Coller Word
1. Ouvrez Word
2. Créez un nouveau document
3. Copiez le contenu de `docs/PROJECT_REPORT_FR.md`
